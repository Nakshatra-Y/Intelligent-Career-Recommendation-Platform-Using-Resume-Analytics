# Intelligent Career Recommendation Platform Using Resume Analytics

## 📌 Project Overview

The Intelligent Career Recommendation Platform is an AI-powered application designed to enhance job-seeking efficiency using Large Language Models (LLMs) and automated job data extraction.

The system analyzes resumes, identifies strengths and weaknesses, generates AI-powered insights, and recommends personalized job opportunities based on user profiles.

The platform is built with a user-friendly Streamlit web interface to provide actionable insights and career guidance.

---

## 🎯 Problem Statement

Choosing the right career path and identifying suitable job opportunities is challenging for job seekers. Many candidates are unaware of:

- Strengths and weaknesses in their resume
- Missing skills required for target roles
- Relevant job opportunities aligned with their profile

This platform solves these challenges using AI-based resume analytics and intelligent job matching.

## 🧩 Modules Implemented

### 1️⃣ User Registration & Resume Analysis Module
- User account creation system
- Resume upload functionality
- Resume parsing and summary generation using LLM
- Secure user data management

### 2️⃣ AI-Powered Resume Insights Module
- Resume analysis using OpenAI / LLM models
- Identification of:
  - Strengths
  - Weaknesses
  - Skill gaps
- Personalized recommendations for improvement
- Interactive Streamlit dashboard for visualization

### 3️⃣ LinkedIn Scraping & Job Recommendation Module
- Job scraping using Selenium
- Extracted job details:
  - Company Name
  - Job Title
  - Location
  - Job URL
  - Job Description
- Matching job listings with user resume profile
- Personalized job recommendations

### 4️⃣ Testing, Review & Documentation
- System functionality testing
- Security review
- Bug fixing and optimization
- Complete technical documentation
